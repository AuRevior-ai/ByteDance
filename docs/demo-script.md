# 3 分钟 Demo 脚本

## 0:00 - 0:20 项目介绍

AgentHub Studio 不是普通聊天机器人，而是一个 IM 式多 Agent 协作工作台。用户可以单聊 Agent，也可以在群聊里让 Orchestrator 自动拆解任务。

## 0:20 - 0:50 主界面展示

展示左侧会话列表和 Agent 列表，中间聊天流，右侧 Artifact Preview 与 Trace 面板。

## 0:50 - 1:30 多 Agent 群聊协作

输入核心 Demo 任务，展示 Orchestrator 生成 Task Plan，并依次调用 PM、UI Designer、Frontend Engineer、Code Reviewer。

## 1:30 - 2:00 Artifact 预览与 Diff

展示 HTML Artifact 在右侧 iframe 中预览。提出局部修改需求，展示 Diff 卡片和版本历史。

## 2:00 - 2:25 部署状态卡片

发送部署指令，展示部署状态、日志和预览地址。

## 2:25 - 2:45 AI Trace

打开 Trace 面板，展示 Agent 调用顺序、Prompt 摘要、耗时和状态。

## 2:45 - 3:00 总结亮点

强调统一 Agent Adapter、产物闭环、AI 协作记录和可复盘 Trace。
